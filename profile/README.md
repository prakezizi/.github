## Repository Structure

Each course repository follows a standardized structure:

```
course-name/
├── README.md                    # Course overview and syllabus
├── Modul-01/                    # Module 1
│   ├── materi.md               # Theory and concepts
│   ├── contoh-program/         # Example programs
│   ├── penjelasan.md           # Detailed explanations
│   └── latihan/                # Practice exercises
├── Modul-02/                    # Module 2
│   └── ...
└── resources/                   # Additional resources
    ├── referensi.md            # References
    └── tools.md                # Required tools and setup
```

## How to Use

1. **Navigate** to the specific course repository you need
2. **Read** the main README for course overview and requirements
3. **Follow** modules sequentially for structured learning
4. **Practice** with provided example programs
5. **Complete** exercises to reinforce understanding
6. **Refer** to additional resources when needed
